# CS231n 2025 学习笔记（五）：卷积神经网络与卷积层

## 1. 为什么需要卷积神经网络？

一张 CIFAR-10 彩色图像的尺寸为：

```math
3 \times 32 \times 32
```

其中：

- 3：RGB 三个颜色通道
- 32：图像高度
- 32：图像宽度

如果使用全连接神经网络，需要先将图像展平为：

```math
3 \times 32 \times 32 = 3072
```

维向量。

这样做存在两个问题：

1. 图像原有的上下、左右空间结构被破坏；
2. 每个神经元都与所有像素连接，参数量很大。

卷积神经网络 CNN 的核心思想是：

> 只观察图像的局部区域，并在整张图像上重复使用同一组参数。

## 2. 什么是卷积？

假设输入图像大小为：

```math
3 \times 32 \times 32
```

使用一个大小为：

```math
3 \times 5 \times 5
```

的滤波器。

滤波器覆盖输入图像中的一个局部区域，并计算：

```math
z = \sum X \odot W + b
```

其中：

- `X`：当前局部图像区域
- `W`：滤波器权重
- `b`：偏置
- `⊙`：对应位置相乘
- `z`：当前位置的输出值

因为：

```math
3 \times 5 \times 5 = 75
```

所以一次卷积可以理解为：

> 将 75 个输入值与 75 个滤波器权重对应相乘并求和，最后加上偏置。

一个滤波器在一个位置只输出一个数。

当滤波器在整张图像上滑动后，就会得到一张二维特征图。

> 深度学习框架中实际执行的通常是互相关运算，即卷积核不翻转，但习惯上仍称为卷积。

## 3. 为什么滤波器深度等于输入通道数？

输入图像有 RGB 三个通道，因此滤波器必须同时覆盖三个通道。

输入图像：

```math
3 \times 32 \times 32
```

滤波器：

```math
3 \times 5 \times 5
```

可以把一个滤波器看成三个二维切片：

```math
W = [W_R,\ W_G,\ W_B]
```

每个切片的大小为：

```math
5 \times 5
```

当前位置的卷积结果为：

```math
z =
\sum X_R \odot W_R
+
\sum X_G \odot W_G
+
\sum X_B \odot W_B
+
b
```

三个通道的计算结果相加，最终只得到一个数。

因此：

> 滤波器的空间尺寸可以较小，但深度必须覆盖输入的全部通道。

## 4. 不同滤波器之间有什么差异？

假设卷积层中有 6 个滤波器，每个滤波器大小都是：

```math
3 \times 5 \times 5
```

它们的形状相同，但内部权重不同：

```math
W_1 \neq W_2 \neq \cdots \neq W_6
```

因此，不同滤波器会对不同的图像特征产生较强响应。

例如：

- 滤波器 1：检测垂直边缘
- 滤波器 2：检测水平边缘
- 滤波器 3：检测斜向边缘
- 滤波器 4：检测颜色变化
- 滤波器 5：检测纹理
- 滤波器 6：检测局部形状

检测垂直边缘的滤波器可能类似：

```math
\begin{bmatrix}
-1 & 0 & 1 \\
-1 & 0 & 1 \\
-1 & 0 & 1
\end{bmatrix}
```

检测水平边缘的滤波器可能类似：

```math
\begin{bmatrix}
-1 & -1 & -1 \\
0 & 0 & 0 \\
1 & 1 & 1
\end{bmatrix}
```

它们尺寸相同，但权重排列不同，因此提取的特征不同。

这些滤波器通常不是人工设计的，而是通过训练自动学习得到的：

```math
W \leftarrow W - \eta \frac{\partial L}{\partial W}
```

网络不会提前规定“第一个滤波器必须检测边缘”，而是通过降低损失函数，让不同滤波器逐渐形成不同的特征提取能力。

## 5. 为什么 6 个滤波器产生 6 张特征图？

一个滤波器扫描整张图像后，会产生一张特征图。

对于输入：

```math
3 \times 32 \times 32
```

使用一个 `3×5×5` 滤波器，并设置：

```math
S = 1,\quad P = 0
```

输出一张：

```math
1 \times 28 \times 28
```

的特征图。

如果使用 6 个滤波器，就会得到：

```math
6 \times 28 \times 28
```

的输出。

因此：

```math
\text{输出通道数} = \text{滤波器数量}
```

需要区分：

- 输入通道数决定每个滤波器的深度；
- 滤波器数量决定卷积层的输出通道数。

在 PyTorch 中，卷积层权重的形状为：

```math
C_{\mathrm{out}}
\times
C_{\mathrm{in}}
\times
K_H
\times
K_W
```

本例中的权重形状为：

```math
6 \times 3 \times 5 \times 5
```

## 6. 卷积输出尺寸

设：

- 输入高度和宽度：`H`、`W`
- 卷积核高度和宽度：`K_H`、`K_W`
- 填充大小：`P_H`、`P_W`
- 步幅：`S_H`、`S_W`

输出高度为：

```math
H_{\mathrm{out}}
=
\left\lfloor
\frac{H + 2P_H - K_H}{S_H}
\right\rfloor
+ 1
```

输出宽度为：

```math
W_{\mathrm{out}}
=
\left\lfloor
\frac{W + 2P_W - K_W}{S_W}
\right\rfloor
+ 1
```

最终输出形状为：

```math
C_{\mathrm{out}}
\times
H_{\mathrm{out}}
\times
W_{\mathrm{out}}
```

当卷积核为正方形，并且两个方向使用相同参数时，可以简写为：

```math
W_{\mathrm{out}}
=
\left\lfloor
\frac{W - K + 2P}{S}
\right\rfloor
+ 1
```

## 7. Stride：步幅

Stride 表示卷积核每次移动多少个像素。

### 7.1 Stride = 1

卷积核每次移动一格：

- 扫描较密；
- 输出尺寸较大；
- 保留更多空间信息。

### 7.2 Stride = 2

卷积核每次移动两格：

- 扫描较稀；
- 输出尺寸较小；
- 计算量更低；
- 可能丢失部分空间细节。

例如：

```math
W = 7,\quad K = 3,\quad P = 0,\quad S = 2
```

输出尺寸为：

```math
W_{\mathrm{out}}
=
\left\lfloor
\frac{7 - 3 + 2 \times 0}{2}
\right\rfloor
+ 1
=
3
```

因此：

```math
7 \times 7
\;\xrightarrow[\text{Stride}=2]{3 \times 3\ \text{卷积}}\;
3 \times 3
```

卷积核左上角可以位于：

```math
0,\quad 2,\quad 4
```

水平方向有 3 个位置，垂直方向也有 3 个位置，所以输出为：

```math
3 \times 3
```

## 8. Padding：填充

如果不使用 Padding，卷积后的特征图会逐渐缩小。

例如：

```math
7 \times 7
\;\xrightarrow[]{3 \times 3\ \text{卷积}}\;
5 \times 5
```

继续卷积：

```math
5 \times 5
\;\xrightarrow[]{3 \times 3\ \text{卷积}}\;
3 \times 3
```

Padding 的做法是在输入图像边缘补零。

当卷积核大小为奇数，且步幅为 1 时，通常设置：

```math
P = \frac{K - 1}{2}
```

这样可以保持输入和输出的空间尺寸相同。

例如：

```math
K = 3,\quad P = 1,\quad S = 1
```

输出尺寸为：

```math
W_{\mathrm{out}}
=
\frac{W - 3 + 2 \times 1}{1}
+ 1
=
W
```

常见设置如下：

| 卷积核 | Padding | Stride | 输出效果 |
|---|---:|---:|---|
| `3×3` | 1 | 1 | 保持空间尺寸 |
| `5×5` | 2 | 1 | 保持空间尺寸 |
| `1×1` | 0 | 1 | 保持空间尺寸 |
| `3×3` | 1 | 2 | 高度和宽度约减半 |

Padding 不仅可以控制输出尺寸，还可以让图像边缘的像素参与更多次卷积运算。

## 9. 卷积层参数量

卷积层的参数量为：

```math
C_{\mathrm{out}}
\left(
C_{\mathrm{in}} K_H K_W + 1
\right)
```

其中最后的 `1` 表示每个滤波器对应一个偏置。

例如：

- 输入通道数：3
- 滤波器数量：6
- 卷积核大小：`5×5`

每个滤波器的参数量为：

```math
3 \times 5 \times 5 + 1 = 76
```

6 个滤波器共有：

```math
6 \times 76 = 456
```

个可学习参数。

如果卷积层不使用偏置，则参数量为：

```math
C_{\mathrm{out}} C_{\mathrm{in}} K_H K_W
```

需要注意：

> 同一个滤波器在图像的所有位置使用同一组权重，因此滤波器移动次数不会增加参数量。

## 10. 局部连接与参数共享

CNN 参数较少的主要原因有两个。

### 10.1 局部连接

卷积神经元只观察输入图像的一小块区域。

例如，一个 `3×3` 卷积核只观察当前位置周围的 `3×3` 区域，而不是整张图像。

### 10.2 参数共享

同一个滤波器会在图像的所有位置重复使用。

如果某个滤波器能够检测垂直边缘，那么无论垂直边缘出现在图像左侧还是右侧，都可以使用同一个滤波器检测。

参数共享带来两个主要好处：

1. 显著减少参数量；
2. 同一种特征可以在不同位置被检测到。

## 11. 感受野

感受野表示：

> 特征图中的一个元素能够看到原始输入图像中的多大区域。

一层 `3×3` 卷积的感受野为：

```math
3 \times 3
```

两层连续的 `3×3` 卷积，在步幅均为 1 时，感受野为：

```math
5 \times 5
```

三层连续的 `3×3` 卷积，感受野为：

```math
7 \times 7
```

当每层步幅均为 1，卷积核大小均为 `K` 时，经过 `L` 层卷积后的感受野为：

```math
R = 1 + L(K - 1)
```

例如：

```math
L = 3,\quad K = 3
```

则：

```math
R = 1 + 3(3 - 1) = 7
```

网络越深，单个神经元能够观察到的输入区域越大。

CNN 通常会逐层形成以下特征：

```text
像素
  ↓
边缘和颜色
  ↓
纹理
  ↓
局部结构
  ↓
高级语义特征
```

浅层卷积通常提取简单特征，深层卷积则将简单特征组合成更复杂的结构。

## 12. Pooling：池化

池化用于降低特征图的空间尺寸。

最常见的是最大池化 Max Pooling。

假设输入为：

```math
\begin{bmatrix}
1 & 1 & 2 & 4 \\
5 & 6 & 7 & 8 \\
3 & 2 & 1 & 0 \\
1 & 2 & 3 & 4
\end{bmatrix}
```

使用大小为 `2×2`、步幅为 2 的最大池化。

第一个区域为：

```math
\begin{bmatrix}
1 & 1 \\
5 & 6
\end{bmatrix}
```

最大值为 6。

最终输出为：

```math
\begin{bmatrix}
6 & 8 \\
3 & 4
\end{bmatrix}
```

池化层的特点：

1. 通常没有可学习参数；
2. 每个通道独立进行池化；
3. 不改变通道数量；
4. 降低特征图的高度和宽度；
5. 对小范围的位置变化更加不敏感。

例如：

```math
64 \times 32 \times 32
\;\xrightarrow[]{2 \times 2\ \text{Max Pool}}\;
64 \times 16 \times 16
```

通道数仍然为 64，但高度和宽度均减半。

在无填充的情况下，池化输出尺寸为：

```math
W_{\mathrm{out}}
=
\left\lfloor
\frac{W - K}{S}
\right\rfloor
+ 1
```

## 13. 平移等变性

卷积具有平移等变性。

简单来说：

> 输入中的特征发生移动，卷积输出中的响应也会对应移动。

例如，图像中的垂直边缘从左侧移动到右侧，对应的特征响应也会从特征图左侧移动到右侧。

理想情况下，可以表示为：

```math
\operatorname{Conv}(\operatorname{Translate}(X))
=
\operatorname{Translate}(\operatorname{Conv}(X))
```

需要区分：

- 平移等变：输入移动，输出也跟着移动；
- 平移不变：输入移动，最终结果基本不变。

卷积主要提供平移等变性。

池化、下采样和最终分类过程，可以使网络对小范围位置变化更加不敏感。

## 14. 1×1 卷积

假设输入特征图为：

```math
64 \times 56 \times 56
```

使用 32 个 `1×1` 卷积核。

由于输入有 64 个通道，所以每个卷积核的完整尺寸为：

```math
64 \times 1 \times 1
```

输出为：

```math
32 \times 56 \times 56
```

因此，`1×1` 卷积可以：

- 融合不同通道的信息；
- 改变通道数量；
- 减少后续计算量；
- 保持空间尺寸不变。

`1×1` 卷积不是只处理一个数，而是在每个空间位置，对全部输入通道进行一次加权组合。

## 15. 基本 CNN 结构

一个基本卷积神经网络可以写成：

```text
Input
  ↓
Convolution
  ↓
ReLU
  ↓
Pooling
  ↓
Convolution
  ↓
ReLU
  ↓
Fully Connected
  ↓
Class Scores
```

各模块的作用：

- Convolution：提取局部特征；
- ReLU：引入非线性；
- Pooling：降低空间尺寸；
- Fully Connected：综合特征并输出类别分数。

例如：

```math
3 \times 32 \times 32
\rightarrow
6 \times 28 \times 28
\rightarrow
6 \times 14 \times 14
\rightarrow
10
```

最后的 10 表示 10 个类别的预测分数。

## 16. PyTorch 示例

```python
import torch
import torch.nn as nn

# 8 张 RGB 图像，每张图像大小为 32×32
x = torch.randn(8, 3, 32, 32)

conv = nn.Conv2d(
    in_channels=3,
    out_channels=6,
    kernel_size=5,
    stride=1,
    padding=0,
)

y = conv(x)

print(y.shape)
```

输出：

```text
torch.Size([8, 6, 28, 28])
```

各维度含义：

- 8：一个批次中有 8 张图像；
- 6：卷积层包含 6 个滤波器；
- 28：输出特征图高度；
- 28：输出特征图宽度。

PyTorch 中图像张量通常采用：

```math
N \times C \times H \times W
```

其中：

- `N`：批次大小
- `C`：通道数
- `H`：高度
- `W`：宽度

## 17. 本节重点总结

1. 卷积层能够保留图像的空间结构。
2. 一个滤波器扫描整张图像后，产生一张特征图。
3. 滤波器深度必须等于输入通道数。
4. 不同滤波器形状可以相同，但内部权重不同。
5. 不同滤波器能够学习不同的图像特征。
6. 输出通道数等于滤波器数量。
7. Stride 控制卷积核每次移动的距离。
8. Padding 用于控制输出尺寸并保留边缘信息。
9. 卷积通过局部连接和参数共享减少参数量。
10. Pooling 降低空间尺寸，但通常不改变通道数量。
11. 网络层数增加后，感受野逐渐扩大。
12. 浅层通常学习边缘和颜色，深层学习复杂结构和高级语义。

## 一句话总结

> CNN 使用多个权重不同的局部滤波器，在整张图像上共享参数，逐层将原始像素转换为边缘、纹理、局部结构和高级语义特征。
