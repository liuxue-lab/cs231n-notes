# CS231n 2025 学习笔记（一）：计算机视觉与深度学习概述

第一节课没有直接讲卷积神经网络、损失函数或者反向传播，而是先介绍了计算机视觉的发展过程、深度学习为什么能够快速发展，以及这门课程后面将学习哪些内容。

这一节没有太多需要推导的公式，主要作用是建立对计算机视觉的整体认识。

## 1. 什么是计算机视觉？

计算机视觉的目标，是让计算机能够从图像和视频中获取信息，并进一步理解现实世界。

一张 RGB 彩色图像在计算机中，本质上是一个三维数组：

```math
\mathbf{I}\in\mathbb{R}^{H\times W\times 3}
```
其中：

- $H$ 表示图像高度；
- $W$ 表示图像宽度；
- $3$ 表示红、绿、蓝三个颜色通道。

人看到一张图片时，可以直接识别出猫、汽车或者行人，但计算机最开始看到的只是大量像素数值。

计算机视觉要完成的，就是从像素到语义的转换：

```text
像素 → 特征 → 物体 → 场景理解
```

计算机视觉也不只是图像分类，还包括：

- 图像分类；
- 目标检测；
- 语义分割；
- 实例分割；
- 视频理解；
- 三维视觉；
- 图像生成；
- 视觉语言模型；
- 机器人视觉。

## 2. 计算机视觉的发展过程

这部分历史内容比较多，我觉得没有必要记住每一个时间点，只需要了解整体发展路线。

早期的计算机视觉主要依靠人工设计特征。

基本流程是：

```text
图像 → 边缘或关键点 → 人工特征 → 分类器
```

研究人员会先从图像中提取边缘、角点和纹理等信息，再使用分类器完成识别。

比较典型的传统视觉方法包括：

- Canny 边缘检测；
- SIFT 特征；
- Viola-Jones 人脸检测。

这些方法在当时非常重要，但存在一个明显问题：

> 图像特征需要研究人员根据经验手动设计。

而深度学习带来的一个重要变化，就是让神经网络直接从数据中学习特征。

现代计算机视觉的基本流程逐渐变成：

```text
原始图像 → 神经网络 → 预测结果
```

传统方法通常需要把特征提取和分类分开设计，而深度学习可以通过端到端训练，同时学习特征和分类规则。

## 3. 神经网络的发展

神经网络并不是最近几年才出现的，它的发展同样经历了很长时间。

### 3.1 感知机

最早的感知机可以看作一个简单的线性分类器：

```math
f(\mathbf{x})
=
\mathrm{sign}
\left(
\mathbf{w}^{T}\mathbf{x}+b
\right)
```
其中：

- $\mathbf{x}$ 是输入特征；
- $\mathbf{w}$ 是模型权重；
- $b$ 是偏置项；
- $\mathrm{sign}(\cdot)$ 是符号函数。

感知机首先计算输入特征的加权和：

```math
z=\mathbf{w}^{T}\mathbf{x}+b
```
然后根据 $z$ 的正负判断样本属于哪个类别。

单层感知机只能处理线性可分问题，无法解决 XOR 这样的非线性分类问题。

### 3.2 多层神经网络与反向传播

后来出现了多层神经网络和反向传播算法，使神经网络能够训练多个网络层。

对于网络参数 $W$，梯度下降的基本更新方式为：

```math
W
\leftarrow
W-\eta\frac{\partial L}{\partial W}
```
其中：

- $L$ 是损失函数；
- $\eta$ 是学习率；
- $\frac{\partial L}{\partial W}$ 是损失函数对参数的梯度。

梯度表示损失函数增长最快的方向，因此沿着梯度的反方向更新参数，可以使损失函数逐渐减小。

反向传播的作用，就是利用链式法则，高效计算神经网络中各个参数的梯度。

可以简单理解为：

```text
前向传播：输入 → 网络计算 → 预测结果 → 损失

反向传播：损失 → 计算梯度 → 更新参数
```

### 3.3 LeNet

1998 年左右出现的 LeNet，已经具有卷积神经网络的基本结构：

```text
输入图像 → 卷积 → 池化 → 卷积 → 池化 → 全连接层 → 分类结果
```

不过，由于当时数据量较少、计算能力有限，神经网络并没有立刻成为计算机视觉的主流方法。

## 4. 深度学习为什么在 2012 年后快速发展？

这节课比较重要的一个结论是，深度学习的成功依赖三个条件：

```text
数据 + 算法 + 计算能力
```

也可以表示为：

> **深度学习的发展 = 数据 + 算法 + 计算能力**

### 4.1 数据

以 ImageNet 为代表的大规模数据集，为深度神经网络提供了足够多的训练样本。

早期数据集规模较小，复杂模型很容易出现过拟合。大规模数据集出现以后，深层网络才有机会学习到更加丰富的视觉特征，例如：

- 边缘；
- 纹理；
- 局部形状；
- 物体部件；
- 完整物体；
- 场景语义。

### 4.2 算法

随着研究不断发展，神经网络中的很多方法逐渐成熟，例如：

- 激活函数；
- 参数初始化；
- 优化算法；
- 正则化方法；
- 数据增强；
- 归一化方法；
- 更深的网络结构。

这些方法让深层神经网络的训练更加稳定，也提高了模型的泛化能力。

### 4.3 计算能力

神经网络训练需要进行大量矩阵乘法和卷积运算。

GPU 具有较强的并行计算能力，非常适合执行这类运算，因此可以大幅提高模型的训练速度。

我对这一部分的理解是：

> 神经网络的基本思想很早就已经出现了，但只有当数据、算法和算力同时成熟后，深度学习才真正发展起来。

## 5. AlexNet 为什么重要？

2012 年，AlexNet 在 ImageNet 图像分类比赛中取得了明显领先的成绩。

AlexNet 的主要特点包括：

- 使用更深的卷积神经网络；
- 使用 GPU 进行训练；
- 使用 ReLU 激活函数；
- 使用数据增强；
- 使用 Dropout 减少过拟合。

AlexNet 最重要的意义，不只是提高了图像分类的准确率，而是证明了：

> 使用大规模数据训练深度神经网络，比人工设计图像特征更有潜力。

从这以后，计算机视觉的主流方法逐渐从：

```text
人工特征 → 传统分类器
```

转向：

```text
深度神经网络 → 端到端学习
```

AlexNet 可以看作深度学习进入计算机视觉主流的重要转折点。

## 6. 计算机视觉中的主要任务

### 6.1 图像分类

图像分类的任务是输入一张图像，然后输出一个类别。

```text
图像 → 类别
```

例如，判断图像中是猫、狗还是汽车。

图像分类一般只判断图像中主要是什么，不需要给出物体的具体位置。

### 6.2 目标检测

目标检测不仅要判断图像中有什么，还需要找到物体所在的位置。

```text
图像 → 物体类别 + 边界框
```

例如，一张图像中同时包含行人、汽车和自行车，目标检测模型需要分别识别这些物体，并用矩形框标出它们的位置。

### 6.3 语义分割

语义分割需要对图像中的每一个像素进行分类。

例如，将图像中的像素分别标记为：

- 天空；
- 道路；
- 汽车；
- 行人；
- 建筑物。

语义分割只区分像素所属的类别，不区分同一类别中的不同物体。

例如，图像中有三个人，语义分割会把他们都标记为“人”。

### 6.4 实例分割

实例分割不仅需要判断每一个像素属于什么类别，还要区分同一类别中的不同物体。

例如，图像中有三个人，实例分割需要分别标出这三个人各自对应的像素区域。

可以简单理解为：

```text
实例分割 = 目标检测 + 像素级分割
```

目标检测给出物体的类别和大致位置，实例分割进一步给出每个物体准确的像素区域。

### 6.5 视频理解

视频比单张图像多了时间信息。

模型除了需要理解每一帧中出现了什么，还需要分析不同帧之间的关系。

常见的视频理解任务包括：

- 动作识别；
- 行为识别；
- 视频事件理解；
- 视频问答；
- 时序动作定位。

## 7. 现代计算机视觉不只有 CNN

CNN 长期以来都是计算机视觉中的主要模型，因为卷积操作非常适合提取图像中的局部特征。

卷积神经网络具有两个重要特点：

- 局部连接；
- 权重共享。

### 7.1 局部连接

局部连接表示卷积核每次只观察图像中的一小块区域。

例如，一个 $3\times3$ 的卷积核，每次只处理输入图像中的一个 $3\times3$ 局部区域。

### 7.2 权重共享

权重共享表示同一个卷积核会在整张图像上重复使用。

这样既可以在图像的不同位置检测相似特征，又可以减少模型参数量。

### 7.3 Transformer

现代计算机视觉已经不再只有 CNN。

Transformer 通过注意力机制建模不同位置之间的关系。

缩放点积注意力可以表示为：

```math
\mathrm{Attention}(Q,K,V)
=
\mathrm{softmax}
\left(
\frac{QK^{T}}{\sqrt{d_k}}
\right)V
```
其中：

- $Q$ 表示查询矩阵；
- $K$ 表示键矩阵；
- $V$ 表示值矩阵；
- $d_k$ 表示键向量的维度。

在视觉任务中，可以把一张图像划分成多个图像块，然后像处理文本序列一样处理这些图像块：

```text
图像 → 图像块序列 → Transformer → 预测结果
```

目前，Transformer 已经广泛应用于：

- 图像分类；
- 目标检测；
- 图像分割；
- 视频理解；
- 视觉语言模型；
- 多模态大模型。

后面学习 CS231n 时，需要重点理解 CNN 和 Transformer 这两类模型。

## 9. 生成模型与视觉语言模型

### 9.1 判别模型

传统的计算机视觉模型，主要完成从图像到标签的映射：

```text
图像 → 标签
```

例如：

```text
猫的图像 → “猫”
```

### 9.2 生成模型

生成模型可以根据文字或者其他条件生成新的图像：

```text
文字或其他条件 → 图像
```

例如：

```text
“一辆行驶在雪地中的红色汽车” → 对应图像
```

### 9.3 扩散模型

扩散模型是目前比较重要的一类生成模型。

它的基本思想是先不断向图像中加入噪声，再训练模型学习反向去噪过程。

正向加噪过程可以表示为：

```math
q(\mathbf{x}_t\mid\mathbf{x}_{t-1})
=
\mathcal{N}
\left(
\mathbf{x}_t;
\sqrt{1-\beta_t}\mathbf{x}_{t-1},
\beta_t\mathbf{I}
\right)
```
其中：

- $\mathbf{x}_{t-1}$ 是上一个时刻的图像；
- $\mathbf{x}_t$ 是加入噪声后的图像；
- $\beta_t$ 控制当前步骤加入的噪声强度；
- $\mathcal{N}$ 表示高斯分布。

反向去噪过程可以表示为：

```math
p_{\theta}(\mathbf{x}_{t-1}\mid\mathbf{x}_t)
=
\mathcal{N}
\left(
\mathbf{x}_{t-1};
\boldsymbol{\mu}_{\theta}(\mathbf{x}_t,t),
\boldsymbol{\Sigma}_{\theta}(\mathbf{x}_t,t)
\right)
```
模型通过学习反向过程，将噪声图像逐渐恢复为清晰图像。

### 9.4 视觉语言模型

视觉语言模型同时处理图像和自然语言，可以完成：

- 图像描述；
- 图文检索；
- 视觉问答；
- 多模态对话；
- 文本控制的图像生成。

第一节课对这些内容主要是简单介绍，后面的课程还会进一步学习。

## 10. 三维视觉与具身智能

二维图像只是三维世界在平面上的投影。

三维视觉希望从图像或其他传感器数据中获得：

- 深度信息；
- 点云；
- 三维结构；
- 物体姿态；
- 相机位置；
- 三维场景模型。

具身智能则要求模型不仅能够看懂环境，还要能够在环境中进行决策和执行动作。

基本过程可以表示为：

```text
视觉观测
    ↓
环境理解
    ↓
任务规划
    ↓
动作执行
```

### 10.1 VLM 与 VLA 的区别

我对 VLM 和 VLA 区别的理解是：

```text
VLM：视觉 + 语言 → 理解或生成
```

```text
VLA：视觉 + 语言 → 动作
```

例如，给模型一张桌面的图像和一句指令：

```text
“把桌子上的红色杯子拿起来。”
```

VLM 可以：

- 识别红色杯子；
- 描述杯子的位置；
- 回答与当前场景有关的问题。

VLA 则需要进一步：

- 判断机械臂应该如何移动；
- 输出对应的动作；
- 完成实际抓取任务。

因此，学习 VLA 之前，需要先掌握计算机视觉、深度学习、Transformer、视觉语言模型以及机器人控制的基础。

## 11. 计算机视觉还没有被完全解决

虽然目前很多模型在标准数据集上的准确率已经很高，但这并不代表模型真正理解了现实世界。

模型在实际环境中仍然可能受到以下因素影响：

- 光照变化；
- 拍摄角度变化；
- 物体遮挡；
- 图像噪声；
- 数据分布变化；
- 训练集中没有出现过的新物体；
- 对抗样本。

例如，一个模型在训练集和测试集上表现很好，但换到不同的摄像头、不同的天气或者不同的环境后，性能可能会明显下降。

因此：

> 测试集上的准确率高，并不等于模型在真实环境中一定可靠。

除了准确率，还需要考虑：

- 泛化能力；
- 鲁棒性；
- 安全性；
- 数据偏差；
- 隐私；
- 公平性。

对于自动驾驶、医疗影像和机器人等安全关键任务，模型的可靠性尤其重要。

## 12. 第一节课总结

第一节课没有太多需要推导的内容，主要是了解计算机视觉和深度学习的发展背景。

我认为需要重点记住以下几点：

1. 计算机视觉的目标，是让机器从图像和视频中理解现实世界。
2. 传统视觉主要依靠人工设计特征，深度学习则通过数据自动学习特征。
3. 深度学习的快速发展依赖数据、算法和计算能力。
4. AlexNet 是深度学习进入计算机视觉主流的重要转折点。
5. 现代计算机视觉不只有 CNN，也大量使用 Transformer。
6. 计算机视觉正在从图像识别发展到生成模型、VLM、三维视觉和具身智能。
7. 数据集上的准确率高，并不代表模型真正理解了现实世界。

可以用两个公式概括这一节课：

> **深度学习的发展 = 数据 + 算法 + 计算能力**

> **计算机视觉 = 从像素中提取信息并理解世界**
